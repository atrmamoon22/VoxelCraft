# VoxelCraft

VoxelCraft is a simple 3D sandbox inspired by Minecraft, featuring a voxel-based world and basic player movement. It’s a minimal environment built for experimentation with 3D rendering and game mechanics.

## Features
- Voxel-based terrain
- First-person and Third-person player movement
- Basic controls and collision
